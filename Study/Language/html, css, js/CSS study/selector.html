selector

tag - > tag name

  • 해당 태그에 일괄적으로 효과 적용

id -> #

  • 파일에 단 한번만 존재
  • 한 태그에 여러 id 가 중복될 수 없음

class -> .

  • 여러번 사용 가능 : 같은 효과를 주고싶은 태그에 중복 사용
  • 한 태그에 여러 class 사용 가능 : 여러 효과를 중복 적용
