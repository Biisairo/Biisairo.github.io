쿼터니언 회전

사원수를 사용해 특정축을 기준으로 특정각도만큼 회전

오일러 회전을 사용하면 보간을 구하기 어려우며

회전에 계산이 많이 들게 된다

ai + bj + ck + d

i, j, k 는 각각 복소수

각각 복소수의 제곱 = -1

ijk 의 곱 = -1

ij = k, ji = -k

jk = i, jk = -i

ki = j, ik = -j

실수부와 허수부 i 를 축으로 하는 2차원 복소평면에서 값에 i를 곱하면 90도씩 회전하게 되있음

다른 허수부 j 를 사용한 삼원수를 사용하여 3차원 공간을 만들면 i, j 를 기준으로 회전할 수 있음

실수 축을 중심으로 하는 회전이 불가능해짐

허수 k 를 추가한 사원수를 사용
